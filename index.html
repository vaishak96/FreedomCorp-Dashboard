FreedomCorp Sales Dashboard

Overview of FreedomCorp Sales Dashboard

The FreedomCorp Sales Dashboard is designed to provide insights into the performance of our sales teams across various regions. The dashboard visualizes sales data, including revenue, call duration, and units sold, to help executives make informed decisions.

Features of the Dashboard

  • Stacked Area Chart: Displays metrics for revenue, call duration, and units sold, broken down by different teams.
  • Interactive Timeline: Below the main graph, there is a sliding window that users can interact with to filter data by specific date ranges.
  • Donut Chart: Summarizes data by company size within the selected time range.
  • Bar Charts: Show summary data for revenue, call duration, and units sold within the selected time range.

How to Use the Dashboard

  • Date Range Selection: Use the timeline brush below the main graph to select a specific date range. The charts will update automatically to reflect the selected date range.
  • Metric Selection: Use the dropdown menu to switch between different metrics (Revenue, Call Time, Units Sold). The stacked area chart and other visualizations will update based on the selected metric.
  • Team Performance Analysis: Analyze the performance of sales teams in different regions to identify trends and make data-driven decisions.
